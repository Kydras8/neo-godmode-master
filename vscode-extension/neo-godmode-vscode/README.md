# Neo Godmode VS Code Extension

This lightweight extension adds commands to insert the Neo Godmode master prompt and create a new prompt file.

## Commands
- **Neo Godmode: Insert Master Prompt**
- **Neo Godmode: New Prompt File**

## Install
1. Zip this folder and rename the archive to `neo-godmode-vscode.vsix`, or run `vsce package` if you use the VS Code Extension Manager.
2. In VS Code, run **Extensions: Install from VSIX...** and select the `.vsix`.

> Note: The extension expects the repo layout so it can find `prompts/neo-godmode-master-prompt.md` two levels up from the extension folder.
